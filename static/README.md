# static/

## But du répertoire

Ce dossier contient les fichiers statiques servant l'interface web de la
calculatrice (CSS, images, scripts éventuels).

## Fichiers principaux
- `style.css` : styles applicatifs pour la page de la calculatrice.

## Pour les développeurs
- Modifications CSS : éditez `style.css` et rechargez la page. Si vous
  ajoutez des bibliothèques frontend plus complexes, documentez le
  processus de build ici.
